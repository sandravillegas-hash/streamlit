Este proyecto ha sido desarrollado como parte del curso de **Análisis de Datos de Talento Tech (Nivel Integrador)**. El objetivo principal es investigar y visualizar la relación entre el uso de dispositivos digitales y la productividad académica en estudiantes de educación superior.

## 👤 Autoría
Realizado por: **Sandra Villegas** *Analista de Datos en Formación - Talento Tech*

## 📝 Descripción del Proyecto
La aplicación es una herramienta interactiva de **Business Intelligence** que permite a los usuarios:
1. **Explorar el contexto**: A través de una Landing Page que explica la problemática de la distracción digital.
2. **Analizar métricas clave**: Visualización de correlaciones, regresiones lineales y distribuciones estadísticas mediante gráficos de **Seaborn**.
3. **Documentación integrada**: Cada gráfico cuenta con una pestaña de ayuda y documentación técnica para facilitar la interpretación de los datos.

## 📂 Dataset
Los datos se gestionan automáticamente mediante la librería `kagglehub`:
* **Fuente:** [Student Productivity and Digital Distraction Dataset](https://www.kaggle.com/datasets/sehaj1104/student-productivity-and-digital-distraction-dataset)
* **Variables clave:** Puntuación de productividad, horas de estudio, puntaje de distracción digital y segmentación demográfica.
